# JEnD（Jindai Encyclopedia of Dining）

## 概要
JEnD（Jindai Encyclopedia of Dining）は，神奈川大学の学生に向けて，キャンパス周辺の個人飲食店を探しやすくすることを目的として制作したWebアプリケーションです。

コロナ禍以降，学生同士，特に先輩から後輩への交流機会が減少し，大学周辺の学生向け飲食店の情報が口伝いで広まりにくくなりました。そこで，近隣の飲食店情報を整理・発信することで，学生が学食やコンビニ食，チェーン店以外の選択肢にも目を向けられるようにしたいと考え，本プロジェクトに取り組みました。

## 背景
コロナ明けの学生生活では，学食やコンビニ食を利用する学生が多く，大学周辺の個人飲食店を利用する機会が以前より減っていました。  
また，コロナ禍をきっかけに，それまで先輩から後輩へ自然に共有されていた「学生向けのおすすめ飲食店情報」が伝わりにくくなっていました。

私たちのチームは，先輩との交流を通じて地域の飲食店に関する情報を知る機会があり，さらに近隣飲食店への関心も高かったことから，それらの情報を整理して発信することで，より多くの学生に地域の飲食店を知ってもらいたいと考えました。

## 目的
本プロジェクトの目的は，神奈川大学の学生がキャンパス周辺の個人飲食店を知り，利用しやすくすることです。  
特に，混雑した学食やコンビニ食以外で昼食を取りたい学生に対して，新たな選択肢を提示することを目指しました。  
また，利用した学生が周囲の学生にも情報を共有することで，地域の飲食店情報が再び広がっていくことも期待していました。

## 想定ユーザー
- 混雑した学食以外で昼食を取りたい学生
- コンビニ食以外の選択肢を探している学生
- 大学周辺の飲食店を知りたい学生
- 近隣の個人飲食店に興味がある学生

## 主な機能
- タグや条件による店舗検索
- 店舗一覧表示
- 店舗詳細ページの表示
- 店舗画像の表示
- 飲食店のSNSへの遷移

## 利用の流れ
1. トップページから店舗一覧へ進む
2. タグや条件を指定して店舗を絞り込む
3. 気になる店舗の詳細ページを見る
4. 必要に応じて飲食店のSNSへ移動する

## 開発体制
研究室の選考ゼミのプロジェクトとして，4人チームで開発しました。

### 自分の担当
私は主に以下を担当しました。

- サイトの動作やユーザーの利用を想定した機能提案
- ユーザーが使いやすいUIの提案
- 掲載する店舗情報の選定
- Webアプリのコーディングの大部分
- 店舗情報入力作業を効率化するためのテンプレート作成
- サーバーへのコードアップロード
- 公開後の動作テスト

単純な店舗情報入力については，テンプレートを作成し，それをもとに他メンバーも作業できるようにしました。

### 他メンバーの担当
- 取材・店舗情報の統括
- Webデザイン・掲載画像の選定
- SNS（Instagram）投稿との連携

## 工夫した点
本プロジェクトでは，ユーザーの探しやすさと，チーム開発の進めやすさの両方を意識しました。

ユーザー視点では，学生が自分のニーズに合わせて飲食店を探せるように，タグや条件によって絞り込みができる構成を考えました。単に一覧を見るだけでなく，条件をもとに候補を絞ってから詳細を見る流れにすることで，目的に合った店舗を見つけやすくすることを目指しました。

開発面では，私がコーディングの大部分を担当しつつ，繰り返し発生する店舗情報入力作業についてはテンプレート化を行い，他メンバーも作業しやすい形に整理しました。これにより，チーム全体で効率よく情報を追加できるように工夫しました。
